Welcome to the Fish Proteome Map (FPM)

The fish proteome map (FPM) is a community resource for proteomic information related to one of the important food fish, Labeo rohita- rohu. The given information is a representation of mass spectrometric data obtained after shotgun proteomics of 17 different organs of the rohu. Whole data is acquired using high resolution Orbitrap mass analyser. Currently it has the information of 7942 proteins based on the identification using NCBI rohu database. Relative protein expression has been presented across multiple tissues along with the list of protein specially identified in a particular tissue. Additionally, it has the information for peptides identified in each tissue. The detailed peptide information for all tissues of rohu is available on peptideAtlas (www.peptideatlas.org). Mass spectrometric raw data can be obtained from Proteome-Xchange using the identifier PXD015451.

The FPM Portal is dynamic and scalable. This portal is based on Django Framework and front-end is developed using Bootstrap V4
